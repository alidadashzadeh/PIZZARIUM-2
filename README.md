# 🍕 Pizzarium

**Live App:** https://pizzarium.vercel.app

Pizzarium is a full-stack pizza ordering platform built with **Next.js (App Router)**, **Supabase**, and **Stripe**.

It demonstrates production-ready architecture including:

- Server-Side Rendering (SSR) for SEO
- Secure Stripe Checkout + Webhook verification
- Server-validated pricing (never trust the client)
- Supabase authentication with SSR cookies
- Optimistic UI updates with React Query
- Custom pizza builder with dynamic image generation
- Clean UI built with shadcn/ui
- Zustand state management

---

# 🚀 Features

## 🔐 Authentication

- Email & password authentication (Supabase Auth)
- Secure SSR session handling via cookies
- Middleware protected routes
- Forgot password flow
- Persistent login state

---

## 🍕 Signature Pizzas & Drinks (SSR)

- Signature pizzas are fetched on the **server** for SEO
- Drinks are also server-rendered
- Filtering and sorting handled via **URL search params**
- Shareable filter URLs
- Back/forward browser support

Example:

```
/signature-pizzas?category=veggie&sortBy=price&order=asc
```

---

## 🛒 Cart & Checkout (Stripe)

- Stripe Checkout Sessions
- All prices validated server-side from the database
- Order created as `pending`
- Stripe webhook updates order to `paid` and `preparing`
- Success page polls until webhook confirmation (handles server cold starts)
- Cancel page preserves cart

### 🔁 Payment Flow

1. User adds items to cart
2. Server validates prices from DB
3. Stripe session created
4. User completes payment
5. Stripe webhook hits `/api/webhooks/stripe`
6. Order marked as paid
7. Success page confirms and clears cart

# ⚙️ Environment Variables

Create a `.env.local` file:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_KEY=
SUPABASE_SERVICE_ROLE_KEY=

STRIPE_PUBLISHABLE_KEY=
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET=

NEXT_PUBLIC_SITE_URL
```

# 🛠 Local Development

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Stripe Webhook (local)

```bash
stripe listen --forward-to localhost:3000/api/webhooks/stripe
```
